# Engineering Equipment Inventory System

Full-stack web application designed to manage engineering equipment used in laboratories, workshops, and technical environments.

## Features
- User authentication and role-based access
- Equipment inventory management
- Search and filtering of equipment
- Equipment ratings and comments
- Shopping cart functionality
- Order management system
- Admin control panel for equipment and users

## Technologies
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- PostgreSQL
- REST API
- jQuery & Bootstrap

## Purpose
This project focuses on simplifying equipment management by providing a centralized system to track, organize, and manage engineering tools and resources while supporting automated workflows and user access control.
